배경 관련 스타일(background~)

background-color : 배경색을 지정하고자 할 때 사용

얼음에 하는 듣기만 대중을 것이다. 더운지라 만천하의 웅대한 이것은 그리하였는가? 거선의 눈에 위하여서, 꽃이 시들어 그것은 교향악이다. 얼음에 있는 심장의 발휘하기 피가 위하여서, 듣기만 아니더면, 철환하였는가? 것은 용감하고 사라지지 피고, 타오르고 힘있다. 가치를 있는 얼음과 가치를 풍부하게 품었기 작고 것이다.

background-clip : 배경을 적용시키고자하는 범위를 지정할 때 사용

        선택자{
            background-clip : border-box(기본값)/padding-box/content-box;
        }

배경 이미지 테스트

        선택자 {
            background-image : url(이미지경로);
            background-repeat : repeat/repeat-x/repeat-y/no-repeat;
            background-size : auto/contain/cover/px px/%;
            backgournd-position : 좌/우/가운데 중 1 위/아래/가운데중1 px/% %;
            background-attachment : scroll/fixed;
        }
